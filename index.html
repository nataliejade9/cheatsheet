Coding Cheat Sheet

Semantic HTML Tags

Tag Name Description
<address> Address Contact information for a person or organisation>
<article> Article A composition intended to be reused i.e. Article or Blog entry
<aside> Aside Represents a document that is indirectly related to the main content
<code> Code Displays it's content styled to indicate the text is code
<figure> Figure Self-contained content such as; an image and optional <figcaption> element
<figcaption> Figure Caption Contains a caption describing the parent <figure> element
<mark> Mark Text Determines text which is marked or highlighted for reference or notation
<section> Generic Section A standalone section that doesn't have a semantic element (always include heading)
<time> Time (Date) Represents a specific time period (datetime attribute to make machine-readable)
